Service
============
We can add Style files any time we want. Both CSS and SCSS.




Command Usage
-------
To add a new ***Style***  use:
    

Example:
```
     $ yo boom:style file_name
```

This will create:
    > app/css/file_name.css

```
    $ yo boom:style file_name -scss
```

This will create:
    > app/css/file_name.scss
